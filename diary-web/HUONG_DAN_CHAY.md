# Hướng dẫn chạy website nhật ký (siêu dễ cho người mới)

Bạn **không cần biết lập trình** để chạy thử website này.

## Cách 1 (Dễ nhất, chỉ cần mở file)

1. Mở thư mục dự án `BTL_LTDD`.
2. Vào thư mục `diary-web`.
3. Nhấp đúp vào file **`index.html`**.
4. Trình duyệt sẽ mở website ngay.

> Nếu trình duyệt hỏi chọn app để mở, hãy chọn Chrome/Edge/Firefox.

---

## Cách 2 (Khuyên dùng – ổn định hơn)

Nếu máy đã có cài **VS Code**:

1. Mở thư mục `BTL_LTDD` bằng VS Code.
2. Cài extension **Live Server** (nếu chưa có).
3. Mở file `diary-web/index.html`.
4. Bấm nút **Go Live** ở góc dưới phải.
5. Website sẽ tự mở ở trình duyệt.

---

## Tài khoản đăng nhập demo

- Tài khoản 1: `minh`  
  Mật khẩu: `123456`
- Tài khoản 2: `an`  
  Mật khẩu: `123456`

---

## Cách test nhanh trong 1 phút

1. Đăng nhập bằng `minh`.
2. Viết 1 nhật ký và bấm **Lưu nhật ký**.
3. Đăng xuất.
4. Đăng nhập bằng `an`.
5. Kiểm tra: bài của `minh` có xuất hiện trong dòng thời gian không.

Nếu có hiển thị tên tác giả + ngày giờ thì website chạy đúng ✅

---

## Nếu bạn mở mà không thấy gì

- Kiểm tra lại bạn mở đúng file: `diary-web/index.html`.
- Thử đổi sang Chrome hoặc Edge.
- Nhấn `Ctrl + F5` để tải lại trang.
